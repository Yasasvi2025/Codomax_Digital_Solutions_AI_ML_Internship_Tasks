## 📊 Complete Internship Roadmap (Day 1 – Day 13)

| Day / Task | Phase | Description & Key Deliverables |
| --- | --- | --- |
| **Day 1** | Environment Setup | Configured Python, Anaconda/VS Code environment, and created initial tracking workspace. |
| **Day 2** | Python Fundamentals | Practical exercises covering variables, data types, control flow structures, and functions. |
| **Day 3** | NumPy Operations | Vectorized computations, array manipulations, indexing, and statistical operations. |
| **Day 4** | Pandas Data Processing | DataFrame operations, filtering, grouping, aggregation, and tabular data management. |
| **Day 5** | Data Preprocessing | Cleaned raw student dataset by handling missing values, duplicates, and feature scaling. |
| **Day 6** | Exploratory Data Analysis | Created data visualizations (bar charts, line graphs, scatter plots) to study feature correlation. |
| **Day 7** | ML Pipeline Preparation | Formatted feature/target arrays (X, y) and performed train/test splitting (70/30 ratio). |
| **Day 8** | Linear Regression Modeling | Trained a Scikit-Learn `LinearRegression` baseline model on historical performance data. |
| **Day 9** | Score Prediction | Executed forward-pass target array inferences across test validation splits. |
| **Day 10** | Model Performance Evaluation | Evaluated performance loss metrics (MAE = 7.41, RMSE = 7.72, R² = -8.53). |
| **Day 11** | Interactive Application | Built a terminal CLI application (`prediction_inference_app.py`) for live score predictions. |
| **Day 12** | Project Architecture | Developed a dynamic Python organizer script to structure files into enterprise-grade folders. |
| **Day 13** | GitHub Deployment | Documented, finalized, and published the complete production codebase to GitHub. |

## ⚙️ Setup & Local Execution

### 1. Clone the Repository

```bash
git clone https://github.com/Yasasvi2025/Codomax_Digital_Solutions_AI_ML_Internship_Tasks.git
cd Codomax_Digital_Solutions_AI_ML_Internship_Tasks

```

### 2. Install Dependencies

```bash
pip install numpy pandas scikit-learn matplotlib seaborn

```

### 3. Run the Score Prediction App

Launch the interactive terminal application to enter midterm scores and receive predicted final exam results:

```bash
python src/prediction_inference_app.py

```

### 4. Run Workspace Organizer Utility

To re-run the dynamic directory structuring tool:

```bash
python Project_Organizer.py

```
